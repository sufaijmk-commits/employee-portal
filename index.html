🔥 Firebase Configuration Required

Please enter your Firebase configuration to connect to your database.

📋 How to set up Firebase for Eden Perfume:

  1. 1. Create Firebase Project: Go to Firebase Console → Create Project
  2. 2. Enable Firestore: Go to Firestore Database → Create database → Start in test mode
  3. 3. Get Web App Config: Project Settings → Your apps → Add web app → Copy config values
  4. 4. Set Security Rules: Firestore → Rules → Use: allow read, write: if true;
  5. 5. Enter Config Below: Paste the values from your Firebase config object
💡 Tip: For production, update Firestore security rules to restrict access appropriately.
